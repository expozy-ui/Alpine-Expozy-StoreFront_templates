2. Ποια δεδομένα συλλέγουμε για εσάς, για ποιο σκοπό και σε ποια νομική βάση τα επεξεργαζόμαστε

Προσωπικά δεδομένα είναι κάθε πληροφορία που μπορεί να ταυτοποιήσει ένα άτομο. Τα ανώνυμα δεδομένα δεν περιλαμβάνονται σε αυτή την κατηγορία. Επεξεργαζόμαστε τις ακόλουθες κατηγορίες προσωπικών δεδομένων:

• Δεδομένα Επικοινωνίας Περιλαμβάνουν κάθε μήνυμα που μας στέλνετε, είτε μέσω της φόρμας επικοινωνίας στον ιστότοπό μας, είτε μέσω email, μηνύματος ή δημοσίευσης σε κοινωνικά δίκτυα ή οποιουδήποτε άλλου τύπου μηνύματος. Επεξεργαζόμαστε αυτά τα δεδομένα για να επικοινωνούμε μαζί σας, για την τήρηση αρχείων και για τη θεμελίωση, άσκηση ή υπεράσπιση νομικών αξιώσεων. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση είναι η απάντηση στα μηνύματα που μας αποστέλλονται, η τήρηση αρχείων και η θεμελίωση ή υπεράσπιση νομικών αξιώσεων.

• Δεδομένα Πελατών. Περιλαμβάνουν δεδομένα που σχετίζονται με την αγορά αγαθών ή/και υπηρεσιών, όπως το όνομά σας, η διεύθυνση τιμολόγησης, η διεύθυνση αποστολής, η διεύθυνση email, άλλα στοιχεία επικοινωνίας (αριθμός τηλεφώνου), στατιστικά στοιχεία αγορών σας και άλλα. Επεξεργαζόμαστε αυτά τα δεδομένα για να παραδώσουμε τα αγαθά ή/και τις υπηρεσίες που αγοράσατε και να διατηρούμε αρχεία για τέτοιες συναλλαγές. Η νομική βάση για αυτή την επεξεργασία είναι η εκτέλεση σύμβασης μεταξύ εσάς και εμάς ή/και η λήψη μέτρων κατόπιν αιτήματός σας για τη σύναψη τέτοιας σύμβασης.

• Δεδομένα Χρηστών. Περιλαμβάνουν δεδομένα σχετικά με τον τρόπο που χρησιμοποιείτε τον ιστότοπό μας και άλλες διαδικτυακές υπηρεσίες, μαζί με οποιαδήποτε δεδομένα δημοσιεύετε στον ιστότοπό μας ή μέσω άλλων διαδικτυακών υπηρεσιών. Επεξεργαζόμαστε αυτά τα δεδομένα για να διαχειριζόμαστε τον ιστότοπό μας και να διασφαλίζουμε ότι σας παρέχουμε κατάλληλο περιεχόμενο, να εγγυόμαστε την ασφάλειά του, να διατηρούμε αντίγραφα ασφαλείας του ιστότοπού μας ή/και της βάσης δεδομένων και να έχουμε τη δυνατότητα διαχείρισης και διοίκησης του ιστότοπού μας, άλλων διαδικτυακών υπηρεσιών και της επιχείρησής μας. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση μας επιτρέπουν τη σωστή διαχείριση του ιστότοπού μας και της επιχείρησής μας.

• Τεχνικά Δεδομένα. Περιλαμβάνουν δεδομένα που σχετίζονται με τη χρήση του ιστότοπού μας και των διαδικτυακών υπηρεσιών, όπως η διεύθυνση IP σας, τα στοιχεία σύνδεσης, λεπτομέρειες για τον browser σας, η διάρκεια επίσκεψης στις σελίδες του ιστότοπού μας, οι προβολές σελίδων και οι διαδρομές πλοήγησης, λεπτομέρειες για τον αριθμό επισκέψεων στον ιστότοπό μας, οι ρυθμίσεις ζώνης ώρας και άλλες τεχνολογίες των συσκευών που χρησιμοποιείτε για πρόσβαση στον ιστότοπό μας. Η πηγή αυτών των δεδομένων είναι το σύστημα παρακολούθησης αναλυτικών στοιχείων. Επεξεργαζόμαστε αυτά τα δεδομένα για να αναλύσουμε τη χρήση του ιστότοπού μας και άλλων διαδικτυακών υπηρεσιών, να διαχειριστούμε και να προστατεύσουμε την επιχείρησή μας και τον ιστότοπό μας, να παρέχουμε κατάλληλο περιεχόμενο και διαφημίσεις και να κατανοήσουμε την αποτελεσματικότητα των διαφημίσεών μας. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση θα μας επιτρέψουν να διαχειριζόμαστε σωστά τον ιστότοπό μας και την επιχείρησή μας, να αναπτύξουμε την επιχείρησή μας και να καθορίσουμε τη στρατηγική μάρκετινγκ.

• Δεδομένα Μάρκετινγκ. Περιλαμβάνουν δεδομένα που σχετίζονται με τις προτιμήσεις σας για λήψη πληροφοριών μάρκετινγκ από εμάς και τρίτους, καθώς και τον προτιμώμενο τρόπο επικοινωνίας σας. Επεξεργαζόμαστε αυτά τα δεδομένα για να μπορούμε να σας συμπεριλάβουμε στις προωθητικές μας ενέργειες, όπως παιχνίδια, βραβεία και δωρεάν δώρα, να σας παρέχουμε κατάλληλο περιεχόμενο και διαφημίσεις και αντίστοιχα να κατανοήσουμε την αποτελεσματικότητα αυτών των διαφημίσεων. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση θα μας βοηθήσουν να μελετήσουμε πώς οι πελάτες χρησιμοποιούν τα προϊόντα/υπηρεσίες μας, να τα αναπτύξουμε, να αναπτύξουμε την επιχείρησή μας και να καθορίσουμε τη στρατηγική μάρκετινγκ.

• Μπορούμε να χρησιμοποιήσουμε τα Δεδομένα Πελατών, τα Δεδομένα Χρηστών, τα Τεχνικά Δεδομένα και τα Δεδομένα Μάρκετινγκ για να σας παρέχουμε κατάλληλο περιεχόμενο και διαφημίσεις (συμπεριλαμβανομένων διαφημίσεων στο Facebook ή άλλων διαφημίσεων) και αντίστοιχα να κατανοήσουμε την αποτελεσματικότητα των παρεχόμενων διαφημίσεων. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σχετίζονται με την ανάπτυξη της επιχείρησής μας. Επίσης μπορούμε να χρησιμοποιήσουμε αυτά τα δεδομένα για να σας στείλουμε άλλες επικοινωνίες μάρκετινγκ. Η νομική βάση για αυτή την επεξεργασία είναι είτε η συγκατάθεση είτε τα έννομα συμφέροντα (δηλαδή η ανάπτυξη της επιχείρησής μας).

Ευαίσθητα Δεδομένα

Δεν συλλέγουμε κανένα ευαίσθητο δεδομένο για εσάς. Ευαίσθητα είναι τα δεδομένα που σχετίζονται με τη φυλή ή την εθνική σας καταγωγή, τις θρησκευτικές ή φιλοσοφικές πεποιθήσεις, τη σεξουαλική ζωή, τον σεξουαλικό προσανατολισμό, τις πολιτικές απόψεις, τη συμμετοχή σε συνδικαλιστικές οργανώσεις, πληροφορίες για την υγεία σας, γενετικά και βιομετρικά δεδομένα. Δεν συλλέγουμε πληροφορίες για ποινικές καταδίκες και αδικήματα.

Παρακαλούμε σημειώστε - όταν μας απαιτείται να συλλέξουμε προσωπικά δεδομένα βάσει νόμου ή σύμφωνα με τους όρους της σύμβασης μεταξύ μας και δεν μας παρέχετε αυτά τα δεδομένα όταν ζητηθούν, ενδέχεται να μην είμαστε σε θέση να εκτελέσουμε τη σύμβαση (για παράδειγμα να σας παραδώσουμε αγαθά ή υπηρεσίες). Εάν δεν μας παρέχετε τα ζητούμενα δεδομένα, ενδέχεται να χρειαστεί να ακυρώσουμε το προϊόν που παραγγείλατε, αλλά αν το κάνουμε, θα σας ενημερώσουμε εγκαίρως.

Θα χρησιμοποιήσουμε τα προσωπικά σας δεδομένα μόνο για τον σκοπό για τον οποίο συλλέχθηκαν ή εάν είναι απαραίτητο για σκοπό παρόμοιο με τον αρχικό. Για περισσότερες πληροφορίες, παρακαλούμε γράψτε μας στο contacts@website.com. Σε περίπτωση που χρειαστεί να χρησιμοποιήσουμε τα δεδομένα σας για έναν νέο άσχετο σκοπό, θα σας ενημερώσουμε, εξηγώντας τους νομικούς λόγους των ενεργειών μας.

Είναι δυνατόν να επεξεργαστούμε τα προσωπικά σας δεδομένα χωρίς τη γνώση ή τη συγκατάθεσή σας, αλλά μόνο στο πλαίσιο που επιτρέπεται από το νόμο. Δεν πραγματοποιούμε αυτοματοποιημένη λήψη αποφάσεων ή αυτοματοποιημένη κατάρτιση προφίλ.

4. Επικοινωνίες Μάρκετινγκ

Η νομική βάση για την επεξεργασία των προσωπικών σας δεδομένων για σκοπούς επικοινωνιών μάρκετινγκ είναι: η συγκατάθεσή σας ή τα έννομα συμφέροντά μας (για παράδειγμα - για την επέκταση της επιχείρησής μας).

Σύμφωνα με τις νομικές διατάξεις για την προστασία των προσωπικών δεδομένων και τις ηλεκτρονικές επικοινωνίες, μπορούμε να σας στέλνουμε μηνύματα μάρκετινγκ εάν:

(Α) πραγματοποιήσετε αγορά, εγγραφείτε στον ιστότοπό μας ή υποβάλετε ερώτημα σχετικά με τα αγαθά ή τις υπηρεσίες μας; ή
(Β) έχετε συμφωνήσει να λαμβάνετε μηνύματα μάρκετινγκ, και σε αμφότερες τις περιπτώσεις δεν έχετε ρητά αρνηθεί να λαμβάνετε τέτοια μηνύματα. Σύμφωνα με τις νομικές διατάξεις, εάν είστε νομικό πρόσωπο, μπορούμε να σας στέλνουμε email μάρκετινγκ χωρίς τη συγκατάθεσή σας. Σε αυτή την περίπτωση επίσης μπορείτε να εξαιρεθείτε από τη λήψη email μάρκετινγκ από εμάς ανά πάσα στιγμή.

Πριν μοιραστούμε τα προσωπικά σας δεδομένα με τρίτους για τους δικούς τους σκοπούς μάρκετινγκ, θα ζητήσουμε τη ρητή συγκατάθεσή σας.

Μπορείτε να μας ζητήσετε να σταματήσουμε να σας στέλνουμε μηνύματα μάρκετινγκ ανά πάσα στιγμή, συνδεόμενοι στον ιστότοπο και το προφίλ σας, και προσθέτοντας ή αφαιρώντας τα αντίστοιχα πεδία/επιλογές για να διορθώσετε τις προτιμήσεις μάρκετινγκ.

Παρακαλούμε σημειώστε - εάν αρνηθείτε να λαμβάνετε μηνύματα μάρκετινγκ, αυτή η άρνηση δεν αφορά προσωπικά δεδομένα που παρέχονται ως αποτέλεσμα άλλων συναλλαγών όπως αγορές, εγγραφές εγγύησης κ.λπ.

Είναι δυνατόν να μοιραστούμε τα προσωπικά σας δεδομένα με τα ακόλουθα μέρη:

• Παρόχους υπηρεσιών που παρέχουν υπηρεσίες IT και διαχείριση συστημάτων;

• Επαγγελματίες συμβούλους, συμπεριλαμβανομένων δικηγόρων, τραπεζιτών, ελεγκτών και ασφαλιστών;

• Κυβερνητικούς φορείς που απαιτούν από εμάς να αναφέρουμε τις διαδικασίες επεξεργασίας;

• Εταιρείες ταχυμεταφορών;

• Τρίτα μέρη που αποκτούν ή συγχωνεύουν μέρη της επιχείρησής μας ή των περιουσιακών στοιχείων.

Απαιτούμε από όλα τα τρίτα μέρη στα οποία μεταφέρουμε τα δεδομένα σας να τηρούν την ασφάλεια των προσωπικών σας δεδομένων και να τα αντιμετωπίζουν σύμφωνα με το νόμο. Τα τρίτα μέρη μπορούν να επεξεργάζονται τα προσωπικά σας δεδομένα μόνο για συγκεκριμένους καθορισμένους σκοπούς και σύμφωνα με τις οδηγίες μας.

6. Ασφάλεια Δεδομένων

Έχουμε εφαρμόσει μέτρα ασφαλείας για να αποτρέψουμε την τυχαία απώλεια, χρήση, τροποποίηση, αποκάλυψη ή πρόσβαση στα προσωπικά σας δεδομένα χωρίς άδεια. Παρέχουμε πρόσβαση στα προσωπικά σας δεδομένα μόνο σε εκείνους τους υπαλλήλους και συνεργάτες που έχουν επιχειρηματική ανάγκη να χρησιμοποιούν τέτοια δεδομένα. Έχουν λάβει οδηγίες να επεξεργάζονται τα προσωπικά σας δεδομένα σύμφωνα με τις οδηγίες μας και να τηρούν τους κανόνες εμπιστευτικότητας.

Έχουμε διαδικασία για την αντιμετώπιση υποψίας διαρροής πληροφοριών προσωπικών σας δεδομένων και θα σας ενημερώσουμε, καθώς και τις αρμόδιες ρυθμιστικές αρχές, σχετικά με τέτοιου τύπου παραβιάσεις, εάν και όταν είναι απαραίτητο.

7. Διατήρηση Δεδομένων

Θα διατηρήσουμε τα προσωπικά σας δεδομένα για όσο χρόνο είναι απαραίτητο για να εκπληρώσουμε τους σκοπούς για τους οποίους τα συλλέξαμε, ή για την ικανοποίηση νομικών, λογιστικών ή απολογιστικών απαιτήσεων.

Όταν αποφασίζουμε για πόσο καιρό θα αποθηκεύσουμε αυτά τα δεδομένα, εξετάζουμε την ποσότητα, τη φύση και την ευαισθησία τους, τον πιθανό κίνδυνο από μη εξουσιοδοτημένη χρήση ή αποκάλυψη και τους σκοπούς της επεξεργασίας, καθώς και το εάν μπορούν να επιτευχθούν με άλλα μέσα και τις νομικές απαιτήσεις.

Παρακαλούμε σημειώστε - οι φορολογικοί νόμοι απαιτούν από εμάς να αποθηκεύουμε βασικές πληροφορίες για τους πελάτες μας (όπως στοιχεία επικοινωνίας, ταυτότητας, οικονομικά δεδομένα και δεδομένα συναλλαγών) για έως και δέκα χρόνια αφότου πάψετε να είστε πελάτης μας.

Υπό ορισμένες συνθήκες μπορούμε να μετατρέψουμε τα προσωπικά σας δεδομένα σε ανώνυμα για σκοπούς έρευνας ή στατιστικής, οπότε σε τέτοιες περιπτώσεις μπορούμε να χρησιμοποιήσουμε αυτές τις πληροφορίες απεριόριστα και χωρίς να σας ειδοποιήσουμε.

Εάν επιθυμείτε να ασκήσετε οποιοδήποτε από τα παραπάνω δικαιώματά σας, παρακαλούμε επικοινωνήστε μαζί μας στο: contacts@website.com

Μπορεί να χρειαστεί να ζητήσουμε από εσάς συγκεκριμένες πληροφορίες για να επιβεβαιώσουμε την ταυτότητά σας και να εγγυηθούμε το δικαίωμά σας για πρόσβαση στα προσωπικά σας δεδομένα (ή για να ασκήσετε οποιοδήποτε από τα άλλα δικαιώματά σας). Θα σας είμαστε ευγνώμονες εάν σε περίπτωση αντίρρησης ή παραπόνου, επικοινωνήσετε πρώτα μαζί μας για να βρούμε τρόπο επίλυσης του προβλήματος.

Δεν ελέγχουμε αυτούς τους ιστότοπους και αντίστοιχα δεν φέρουμε ευθύνη για τις δηλώσεις απορρήτου τους. Όταν εγκαταλείπετε τον ιστότοπό μας, σας συνιστούμε να διαβάσετε την ανακοίνωση απορρήτου κάθε ιστότοπου που επισκέπτεστε.

11. Cookies

Μπορείτε να ρυθμίσετε τον browser σας να απορρίπτει όλα ή ορισμένα cookies ή να σας προειδοποιεί όταν οι ιστότοποι ορίζουν ή έχουν πρόσβαση σε cookies. Εάν απενεργοποιήσετε ή απορρίψετε τα cookies, παρακαλούμε έχετε υπόψη ότι ορισμένα μέρη αυτού του ιστότοπου ενδέχεται να γίνουν απρόσιτα ή να μην λειτουργούν σωστά.
