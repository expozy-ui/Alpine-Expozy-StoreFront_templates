2. Quelles données collectons-nous vous concernant, dans quel but et sur quelle base les traitons-nous

Les données personnelles sont toute information permettant d'identifier une personne. Les données anonymes ne sont pas incluses dans cette catégorie. Nous traitons les catégories suivantes de données personnelles :

• Données de communication Cela inclut tout message que vous nous envoyez, que ce soit via le formulaire de communication de notre site, par e-mail, message ou publication sur les réseaux sociaux ou tout autre type de message envoyé. Nous traitons ces données afin de communiquer avec vous, de tenir des registres et d'établir, exécuter ou défendre des réclamations légales. La base juridique de ce traitement est nos intérêts légitimes, qui dans ce cas sont de répondre aux messages qui nous sont envoyés, de tenir des registres et de mener ou défendre des réclamations légales.

• Données client. Cela inclut les données liées à l'achat de biens et/ou services, telles que votre nom, adresse de facturation, adresse de livraison, adresse e-mail, autres coordonnées (numéro de téléphone), statistiques sur vos achats, etc. Nous traitons ces données pour livrer les biens et/ou services que vous avez achetés et pour tenir des registres de ces transactions. La base juridique de ce traitement est l'exécution d'un contrat entre vous et nous et/ou la prise de mesures à votre demande pour conclure un tel contrat.

• Données utilisateur. Cela inclut les données sur la façon dont vous utilisez notre site Web et d'autres services en ligne, ainsi que toutes les données que vous publiez sur notre site Web ou via d'autres services en ligne. Nous traitons ces données pour gérer notre site Web et nous assurer que nous vous fournissons un contenu approprié, pour garantir sa sécurité, pour maintenir des copies de sauvegarde de notre site Web et/ou base de données et pour avoir la possibilité de gérer et d'administrer notre site Web, d'autres services en ligne et notre entreprise. La base juridique de ce traitement est nos intérêts légitimes, qui dans ce cas nous permettent d'administrer correctement notre site Web et notre entreprise.

• Données techniques. Cela inclut les données liées à l'utilisation de notre site Web et des services en ligne, telles que votre adresse IP, vos données de connexion, les détails de votre navigateur, la durée des visites sur les pages de notre site Web, les pages vues et les chemins de navigation, les détails sur le nombre de visites sur notre site Web, les paramètres de fuseau horaire et d'autres technologies des appareils que vous utilisez pour accéder à notre site Web. La source de ces données provient de notre système de suivi analytique. Nous traitons ces données pour analyser l'utilisation de notre site Web et d'autres services en ligne, pour administrer et protéger notre entreprise et notre site Web, pour fournir un contenu et des publicités appropriés, et pour comprendre l'efficacité de nos publicités. La base juridique de ce traitement est nos intérêts légitimes, qui dans ce cas nous permettront d'administrer correctement notre site Web et notre entreprise, de développer notre activité et de définir notre stratégie marketing.

• Données marketing. Cela inclut les données liées à vos préférences pour recevoir des informations marketing de notre part et de tiers, et votre méthode de communication préférée. Nous traitons ces données afin de pouvoir vous inclure dans nos promotions, telles que des jeux, des récompenses et des cadeaux gratuits, pour vous fournir un contenu et des publicités adéquats, et pour comprendre l'efficacité de ces publicités. La base juridique de ce traitement est nos intérêts légitimes, qui dans ce cas nous aideront à étudier comment les clients utilisent nos produits/services, à les développer, à développer notre activité et à définir notre stratégie marketing.

• Nous pouvons utiliser les Données client, les Données utilisateur, les Données techniques et les Données marketing pour vous fournir un contenu et des publicités adéquats (y compris des publicités Facebook ou d'autres publicités) et pour comprendre l'efficacité des publicités fournies. La base juridique de ce traitement est nos intérêts légitimes, qui sont liés à la croissance de notre entreprise. Nous pouvons également utiliser ces données pour vous envoyer d'autres communications marketing. La base juridique de ce traitement est soit le consentement, soit les intérêts légitimes (à savoir la croissance de notre entreprise).

Données sensibles

Nous ne collectons aucune donnée sensible vous concernant. Les données sensibles sont celles liées à votre race ou origine ethnique, convictions religieuses ou philosophiques, vie sexuelle, orientation sexuelle, opinions politiques, appartenance syndicale, informations sur votre santé, données génétiques et biométriques. Nous ne collectons pas d'informations sur les condamnations pénales et les infractions.

Veuillez noter - lorsque nous sommes tenus de collecter des données personnelles par la loi ou conformément aux termes du contrat entre nous et que vous ne nous fournissez pas ces données sur demande, nous pourrions ne pas être en mesure d'exécuter le contrat (par exemple, vous livrer des biens ou des services). Si vous ne nous fournissez pas les données demandées, nous pourrions devoir annuler le produit que vous avez commandé, mais si nous le faisons, nous vous en informerons à temps.

Nous n'utiliserons vos données personnelles que dans le but pour lequel elles ont été collectées ou si cela est nécessaire à des fins similaires à l'objectif principal. Pour plus d'informations, veuillez nous écrire à contacts@website.com. Si nous devons utiliser vos données à des fins nouvelles non liées, nous vous en informerons en exposant les raisons juridiques de nos actions.

Nous pouvons traiter vos données personnelles à votre insu ou sans votre consentement, mais uniquement dans les limites autorisées par la loi. Nous n'effectuons pas de prise de décision automatisée ni de profilage automatisé.

4. Communications marketing

La base juridique du traitement de vos données personnelles à des fins de communications marketing est : votre consentement ou nos intérêts légitimes (par exemple - pour développer notre entreprise).

Conformément aux dispositions légales sur la protection des données personnelles et les communications électroniques, nous pouvons vous envoyer des communications marketing si :

(A) vous effectuez un achat, vous inscrivez sur notre site ou nous faites une demande concernant nos biens ou services ; ou
(B) vous avez accepté de recevoir des communications marketing, et dans les deux cas, vous n'avez pas expressément refusé de recevoir de telles communications. Conformément aux dispositions légales, si vous êtes une personne morale, nous pouvons vous envoyer des e-mails marketing sans votre consentement. Dans ce cas également, vous pouvez refuser de recevoir des e-mails marketing de notre part à tout moment.

Avant de partager vos données personnelles avec des tiers à leurs fins marketing, nous vous demanderons votre consentement explicite.

Vous pouvez nous demander d'arrêter de vous envoyer des communications marketing à tout moment en vous connectant au site Web et à votre profil, et en ajoutant ou en supprimant les champs/options appropriés pour corriger vos préférences marketing.

Veuillez noter - si vous refusez de recevoir des communications marketing, ce refus ne s'applique pas aux données personnelles fournies à la suite d'autres transactions telles que des achats, des enregistrements de garantie, etc.

Nous pouvons partager vos données personnelles avec les parties suivantes :

• Des prestataires de services qui fournissent des services informatiques et d'administration système ;

• Des conseillers professionnels, y compris des avocats, des banquiers, des auditeurs et des assureurs ;

• Des autorités gouvernementales qui nous obligent à rendre compte des processus de traitement ;

• Des sociétés de messagerie ;

• Des tiers qui acquièrent ou fusionnent des parties de notre entreprise ou de nos actifs.

Nous exigeons de tous les tiers auxquels nous transférons vos données qu'ils respectent la sécurité de vos données personnelles et qu'ils les traitent conformément à la loi. Les tiers ne peuvent traiter vos données personnelles qu'à des fins spécifiquement définies et conformément à nos instructions.

6. Sécurité des données

Nous avons mis en place des mesures de sécurité pour empêcher la perte accidentelle, l'utilisation, la modification, la divulgation ou l'accès à vos données personnelles sans autorisation. Nous ne donnons accès à vos données personnelles qu'aux employés et partenaires qui ont un besoin professionnel d'utiliser ces données. Ils sont chargés de traiter vos données personnelles selon nos instructions et de respecter les règles de confidentialité.

Nous avons une procédure pour traiter les soupçons de fuite d'informations de vos données personnelles et nous vous informerons, ainsi que les régulateurs compétents, de ce type de violations si et quand cela est nécessaire.

7. Conservation des données

Nous conserverons vos données personnelles aussi longtemps que nécessaire pour atteindre les objectifs pour lesquels nous les avons collectées ou pour satisfaire aux exigences légales, comptables ou de rapport.

Lorsque nous décidons de la durée de conservation de ces données, nous prenons en compte leur quantité, leur nature et leur sensibilité, le risque potentiel d'utilisation ou de divulgation non autorisée et les finalités du traitement, ainsi que la possibilité d'atteindre ces objectifs par d'autres moyens et les exigences légales.

Veuillez noter - les lois fiscales nous obligent à conserver les informations de base sur nos clients (telles que les coordonnées, l'identité, les données financières et les données de transaction) pendant une période allant jusqu'à dix ans après que vous cessez d'être nos clients.

Dans certaines circonstances, nous pouvons rendre vos données personnelles anonymes à des fins de recherche ou de statistiques, auquel cas nous pouvons utiliser ces informations indéfiniment et sans vous en informer.

Si vous souhaitez exercer l'un de vos droits mentionnés ci-dessus, veuillez nous contacter à : contacts@website.com

L'accès à vos données personnelles (ou l'exercice de l'un de vos autres droits) n'est pas payant. Cependant, la loi nous permet d'imposer des frais raisonnables si votre demande est manifestement infondée, répétitive ou excessive, ou n'est pas compatible avec votre demande dans les circonstances ci-dessus.

Nous pourrions avoir besoin de vous demander des informations spécifiques pour confirmer votre identité et garantir votre droit d'accès à vos données personnelles (ou d'exercer l'un de vos autres droits). Cette mesure de sécurité garantit que vos données personnelles ne sont pas divulguées à des personnes qui n'ont pas le droit de les recevoir. Nous pouvons également vous contacter pour vous demander des informations supplémentaires concernant votre demande, afin d'accélérer notre réponse.

Le délai de réponse à toutes les demandes légitimes est d'un mois. Si votre demande est particulièrement complexe ou si vous avez fait plusieurs demandes et que nous ne pouvons pas vous répondre dans le mois, nous vous en informerons.

Si vous n'êtes pas satisfait de la façon dont vos données sont collectées et utilisées, vous avez le droit de déposer une plainte auprès de la Commission nationale de l'informatique et des libertés (CNIL). Nous vous serions reconnaissants de nous contacter d'abord en cas d'objection ou de plainte, afin de trouver un moyen de résoudre le problème.
