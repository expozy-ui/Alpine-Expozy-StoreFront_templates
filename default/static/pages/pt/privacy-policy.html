2. Que dados recolhemos sobre si, para que finalidade e com que base os processamos

Dados pessoais são qualquer informação que possa identificar uma pessoa. Dados anónimos não estão incluídos nesta categoria. Processamos as seguintes categorias de dados pessoais:

• Dados de comunicação Incluem qualquer mensagem que nos envie, seja através do formulário de comunicação no nosso website, por email, mensagem ou publicação em redes sociais ou qualquer outro tipo de mensagem enviada. Processamos estes dados com o objetivo de comunicar consigo, manter registos e estabelecer, executar ou defender reivindicações legais. A base legal para este processamento são os nossos interesses legítimos, que neste caso são responder às mensagens enviadas para nós, manter registos e conduzir ou defender reivindicações legais.

• Dados do cliente. Incluem dados relacionados com a compra de bens e/ou serviços, como o seu nome, endereço de faturação, endereço de entrega, endereço de email, outros dados de contacto (número de telefone), estatísticas sobre as suas compras e outros. Processamos estes dados para entregar os bens e/ou serviços que comprou e para manter registos de tais transações. A base legal para este processamento é a execução de um contrato entre si e nós e/ou a tomada de medidas a seu pedido para celebrar tal contrato.

• Dados do utilizador. Incluem dados sobre como utiliza o nosso website e outros serviços online, juntamente com quaisquer dados que publique no nosso website ou através de outros serviços online. Processamos estes dados para gerir o nosso website e garantir que lhe fornecemos conteúdo apropriado, para garantir a sua segurança, para manter cópias de segurança do nosso website e/ou base de dados, e para poder gerir e administrar o nosso website, outros serviços online e negócio. A base legal para este processamento são os nossos interesses legítimos, que neste caso nos permitem administrar corretamente o nosso website e negócio.

• Dados técnicos. Incluem dados relacionados com a utilização do nosso website e serviços online, como o seu endereço IP, dados de login, detalhes sobre o seu navegador, duração da visita às páginas do nosso website, visualizações de páginas e caminhos de navegação, detalhes sobre o número de visitas ao nosso website, configurações de fuso horário e outras tecnologias dos dispositivos que utiliza para aceder ao nosso website. A fonte destes dados é o nosso sistema de rastreamento analítico. Processamos estes dados para analisar a utilização do nosso website e outros serviços online, para administrar e proteger o nosso negócio e website, para fornecer conteúdo e publicidade apropriados, e para compreender a eficácia da nossa publicidade. A base legal para este processamento são os nossos interesses legítimos, que neste caso nos permitirão administrar corretamente o nosso website e negócio, fazer crescer o nosso negócio e determinar a nossa estratégia de marketing.

• Dados de marketing. Incluem dados relacionados com as suas preferências para receber informações de marketing de nós e de terceiros, e o seu método de comunicação preferido. Processamos estes dados para que possamos incluí-lo nas nossas promoções, como jogos, prémios e presentes gratuitos, fornecer-lhe conteúdo e publicidade adequados, e assim compreender a eficácia destas publicidades. A base legal para este processamento são os nossos interesses legítimos, que neste caso nos ajudarão a estudar como os clientes utilizam os nossos produtos/serviços, desenvolvê-los, fazer crescer o nosso negócio e determinar a nossa estratégia de marketing.

• Podemos utilizar Dados do cliente, Dados do utilizador, Dados técnicos e Dados de marketing para lhe fornecer conteúdo e publicidade adequados (incluindo anúncios do Facebook ou outros anúncios) e assim compreender a eficácia da publicidade fornecida. A base legal para este processamento são os nossos interesses legítimos, que estão relacionados com o crescimento do nosso negócio. Também podemos utilizar estes dados para lhe enviar outras comunicações de marketing. A base legal para este processamento é o consentimento ou os interesses legítimos (nomeadamente o crescimento do nosso negócio).

Dados sensíveis

Não recolhemos quaisquer dados sensíveis sobre si. Dados sensíveis incluem dados relacionados com a sua raça ou origem étnica, crenças religiosas ou filosóficas, vida sexual, orientação sexual, opiniões políticas, filiação sindical, informações sobre a sua saúde, dados genéticos e biométricos. Não recolhemos informações sobre condenações criminais e infrações.

Por favor note - quando somos obrigados a recolher dados pessoais por lei ou nos termos do contrato entre nós e você não nos fornece estes dados quando solicitados, podemos não ser capazes de executar o contrato (por exemplo, entregar-lhe bens ou serviços). Se não nos fornecer os dados solicitados, podemos ter de cancelar o produto que encomendou, mas se o fizermos, notificá-lo-emos atempadamente.

Utilizaremos os seus dados pessoais apenas para a finalidade para a qual foram recolhidos ou se necessário para uma finalidade semelhante à principal. Para mais informações, escreva-nos para contacts@website.com. Se precisarmos de utilizar os seus dados para uma nova finalidade não relacionada, notificá-lo-emos, indicando as razões legais para as nossas ações.

Podemos processar os seus dados pessoais sem o seu conhecimento ou consentimento, mas apenas dentro dos limites permitidos por lei. Não realizamos tomada de decisões automatizada nem criação de perfis automatizada.

4. Comunicações de marketing

A base legal para o processamento dos seus dados pessoais para fins de comunicações de marketing é: o seu consentimento ou os nossos interesses legítimos (por exemplo - para expandir o nosso negócio).

De acordo com as disposições legais sobre a proteção de dados pessoais e comunicações eletrónicas, podemos enviar-lhe mensagens de marketing se:

(A) efetuar uma compra, registar-se no nosso website ou fazer uma consulta sobre os nossos bens ou serviços; ou
(B) concordou em receber mensagens de marketing, e em ambos os casos não recusou expressamente receber tais mensagens. De acordo com os regulamentos legais, se for uma pessoa coletiva, podemos enviar-lhe emails de marketing sem o seu consentimento. Neste caso, também pode optar por não receber emails de marketing da nossa parte a qualquer momento.

Antes de partilhar os seus dados pessoais com terceiros para os seus fins de marketing, pediremos o seu consentimento explícito.

Pode solicitar-nos que paremos de lhe enviar mensagens de marketing a qualquer momento iniciando sessão no website e no seu perfil, e adicionando ou cancelando os campos/opções correspondentes para ajustar as suas preferências de marketing.

Por favor note - se recusar receber mensagens de marketing, esta recusa não se aplica a dados pessoais fornecidos como resultado de outras transações como compras, registos de garantia, etc.

Podemos partilhar os seus dados pessoais com as seguintes partes:

6. Segurança dos dados

Implementámos medidas de segurança para prevenir a perda, uso, alteração, divulgação ou acesso não autorizado acidental aos seus dados pessoais. Fornecemos acesso aos seus dados pessoais apenas àqueles funcionários e parceiros que têm uma necessidade comercial de utilizar tais dados. São instruídos a processar os seus dados pessoais de acordo com as nossas instruções e a cumprir as regras de confidencialidade.

Temos um procedimento em vigor para lidar com suspeitas de violações dos seus dados pessoais e notificá-lo-emos, bem como aos reguladores relevantes, de tais violações se e quando necessário.

7. Retenção de dados

Reteremos os seus dados pessoais pelo tempo necessário para cumprir os fins para os quais os recolhemos, ou para satisfazer requisitos legais, contabilísticos ou de reporte.

Ao decidir quanto tempo armazenar estes dados, consideramos a sua quantidade, natureza e sensibilidade, o risco potencial de uso ou divulgação não autorizados e os fins do processamento, bem como se podem ser alcançados por outros meios e os requisitos legais.

Por favor note - as leis fiscais exigem que retenhamos informações básicas sobre os nossos clientes (como dados de contacto, identidade, dados financeiros e dados de transações) por até dez anos após deixar de ser nosso cliente.

Em certas circunstâncias, podemos anonimizar os seus dados pessoais para fins de pesquisa ou estatísticos, caso em que podemos utilizar esta informação indefinidamente e sem notificá-lo.

Se desejar exercer algum dos direitos acima, contacte-nos em: contacts@website.com

Agradeceríamos se, em caso de objeção ou reclamação, primeiro nos contactasse para que possamos encontrar uma forma de resolver o problema.
