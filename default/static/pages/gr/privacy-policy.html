Επεξεργαζόμαστε αυτά τα δεδομένα για να επικοινωνήσουμε μαζί σας, να τηρούμε αρχεία και να θεμελιώσουμε, εκτελέσουμε ή υπερασπιστούμε νομικές αξιώσεις. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση είναι η απάντηση σε μηνύματα που μας αποστέλλονται, η τήρηση αρχείων και η διεξαγωγή ή υπεράσπιση νομικών αξιώσεων.

• Δεδομένα Πελάτη. Περιλαμβάνουν δεδομένα σχετικά με την αγορά αγαθών και/ή υπηρεσιών, όπως το όνομά σας, τη διεύθυνση χρέωσης, τη διεύθυνση παράδοσης, τη διεύθυνση email, άλλα στοιχεία επικοινωνίας (αριθμός τηλεφώνου), στατιστικά στοιχεία σχετικά με τις αγορές σας και άλλα. Επεξεργαζόμαστε αυτά τα δεδομένα για να παραδώσουμε τα αγαθά και/ή τις υπηρεσίες που αγοράσατε και να τηρούμε αρχεία τέτοιων συναλλαγών. Η νομική βάση για αυτή την επεξεργασία είναι η εκτέλεση σύμβασης μεταξύ υμών και ημών και/ή η λήψη μέτρων κατόπιν αιτήματός σας για τη σύναψη τέτοιας σύμβασης.

• Δεδομένα Χρήστη. Περιλαμβάνουν δεδομένα σχετικά με τον τρόπο που χρησιμοποιείτε τον ιστότοπό μας και άλλες διαδικτυακές υπηρεσίες, μαζί με οποιαδήποτε δεδομένα δημοσιεύετε στον ιστότοπό μας ή μέσω άλλων διαδικτυακών υπηρεσιών. Επεξεργαζόμαστε αυτά τα δεδομένα για να διαχειριστούμε τον ιστότοπό μας και να διασφαλίσουμε ότι σας παρέχουμε κατάλληλο περιεχόμενο, για να διασφαλίσουμε την ασφάλειά του, να διατηρούμε αντίγραφα ασφαλείας του ιστότοπου και/ή της βάσης δεδομένων μας, και να μπορούμε να διαχειριστούμε και να διοικήσουμε τον ιστότοπό μας, άλλες διαδικτυακές υπηρεσίες και την επιχείρησή μας. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση μας επιτρέπουν να διαχειριστούμε σωστά τον ιστότοπο και την επιχείρησή μας.

• Τεχνικά Δεδομένα. Περιλαμβάνουν δεδομένα σχετικά με τη χρήση του ιστότοπου και των διαδικτυακών υπηρεσιών μας, όπως τη διεύθυνση IP σας, τα δεδομένα σύνδεσης, λεπτομέρειες σχετικά με το πρόγραμμα περιήγησής σας, τη διάρκεια επίσκεψης στις σελίδες του ιστότοπού μας, τις προβολές σελίδων και τις διαδρομές πλοήγησης, λεπτομέρειες σχετικά με τον αριθμό επισκέψεων στον ιστότοπό μας, τις ρυθμίσεις ζώνης ώρας και άλλες τεχνολογίες των συσκευών που χρησιμοποιείτε για πρόσβαση στον ιστότοπό μας. Η πηγή αυτών των δεδομένων είναι το σύστημα παρακολούθησης αναλυτικών στοιχείων μας. Επεξεργαζόμαστε αυτά τα δεδομένα για να αναλύσουμε τη χρήση του ιστότοπου και άλλων διαδικτυακών υπηρεσιών μας, για να διαχειριστούμε και να προστατέψουμε την επιχείρηση και τον ιστότοπό μας, για να παρέχουμε κατάλληλο περιεχόμενο και διαφήμιση, και για να κατανοήσουμε την αποτελεσματικότητα της διαφήμισής μας. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση θα μας επιτρέψουν να διαχειριστούμε σωστά τον ιστότοπο και την επιχείρησή μας, να αναπτύξουμε την επιχείρησή μας και να καθορίσουμε τη στρατηγική μάρκετινγκ μας.

• Δεδομένα Μάρκετινγκ. Περιλαμβάνουν δεδομένα σχετικά με τις προτιμήσεις σας για τη λήψη πληροφοριών μάρκετινγκ από εμάς και τρίτους, και τον προτιμώμενο τρόπο επικοινωνίας σας. Επεξεργαζόμαστε αυτά τα δεδομένα ώστε να μπορούμε να σας συμπεριλάβουμε στις προωθητικές μας ενέργειες, όπως παιχνίδια, βραβεία και δωρεάν δώρα, να σας παρέχουμε κατάλληλο περιεχόμενο και διαφήμιση, και αντίστοιχα να κατανοήσουμε την αποτελεσματικότητα αυτών των διαφημίσεων. Η νομική βάση για αυτή την επεξεργασία είναι τα έννομα συμφέροντά μας, τα οποία σε αυτή την περίπτωση θα μας βοηθήσουν να μελετήσουμε πώς οι πελάτες χρησιμοποιούν τα προϊόντα/υπηρεσίες μας, να τα αναπτύξουμε, να μεγαλώσουμε την επιχείρησή μας και να καθορίσουμε τη στρατηγική μάρκετινγκ μας.

• Ενδέχεται να χρησιμοποιήσουμε Δεδομένα Πελάτη, Δεδομένα Χρήστη, Τεχνικά Δεδομένα και Δεδομένα Μάρκετινγκ για να σας παρέχουμε κατάλληλο περιεχόμενο και διαφήμιση (συμπεριλαμβανομένων διαφημίσεων Facebook ή άλλων διαφημίσεων) και αντίστοιχα να κατανοήσουμε την αποτελεσματικότητα της παρεχόμενης διαφήμισης. Ευαίσθητα Δεδομένα

Δεν συλλέγουμε ευαίσθητα δεδομένα για εσάς. Τα ευαίσθητα δεδομένα περιλαμβάνουν δεδομένα σχετικά με τη φυλή ή την εθνοτική καταγωγή σας, τις θρησκευτικές ή φιλοσοφικές πεποιθήσεις, τη σεξουαλική ζωή, τον σεξουαλικό προσανατολισμό, τις πολιτικές απόψεις, τη συνδικαλιστική ιδιότητα, πληροφορίες σχετικά με την υγεία σας, γενετικά και βιομετρικά δεδομένα. Δεν συλλέγουμε πληροφορίες σχετικά με ποινικές καταδίκες και αδικήματα.

Παρακαλούμε σημειώστε - όταν απαιτείται από το νόμο ή σύμφωνα με τους όρους της σύμβασης μεταξύ μας να συλλέξουμε προσωπικά δεδομένα και δεν μας τα παρέχετε όταν τα ζητάμε, ενδέχεται να μην μπορέσουμε να εκτελέσουμε τη σύμβαση (για παράδειγμα, να σας παραδώσουμε αγαθά ή υπηρεσίες). Εάν δεν μας παρέχετε τα ζητούμενα δεδομένα, ενδέχεται να χρειαστεί να ακυρώσουμε το προϊόν που παραγγείλατε, αλλά αν το κάνουμε, θα σας ειδοποιήσουμε εγκαίρως.

Θα χρησιμοποιήσουμε τα προσωπικά σας δεδομένα μόνο για τον σκοπό για τον οποίο συλλέχθηκαν ή εάν είναι απαραίτητο για έναν σκοπό παρόμοιο με τον κύριο. Για περισσότερες πληροφορίες, γράψτε μας στο contacts@website.com. Εάν χρειαστεί να χρησιμοποιήσουμε τα δεδομένα σας για έναν νέο άσχετο σκοπό, θα σας ειδοποιήσουμε, δηλώνοντας τους νομικούς λόγους των ενεργειών μας.

Ενδέχεται να επεξεργαστούμε τα προσωπικά σας δεδομένα χωρίς τη γνώση ή τη συγκατάθεσή σας, αλλά μόνο εντός των ορίων που επιτρέπει ο νόμος. Δεν εκτελούμε αυτοματοποιημένη λήψη αποφάσεων ή αυτοματοποιημένη κατάρτιση προφίλ.

4. Επικοινωνίες Μάρκετινγκ

Η νομική βάση για την επεξεργασία των προσωπικών σας δεδομένων για επικοινωνίες μάρκετινγκ είναι: η συγκατάθεσή σας ή τα έννομα συμφέροντά μας (για παράδειγμα - για την επέκταση της επιχείρησής μας).

Σύμφωνα με τις νομικές διατάξεις για την προστασία προσωπικών δεδομένων και τις ηλεκτρονικές επικοινωνίες, ενδέχεται να σας στείλουμε μηνύματα μάρκετινγκ εάν:

(Α) πραγματοποιείτε μια αγορά, εγγράφεστε στον ιστότοπό μας ή κάνετε μια ερώτηση σε εμάς σχετικά με τα αγαθά ή τις υπηρεσίες μας· ή
(Β) έχετε συμφωνήσει να λαμβάνετε μηνύματα μάρκετινγκ, και σε αμφότερες τις περιπτώσεις δεν έχετε ρητά αρνηθεί να λαμβάνετε τέτοια μηνύματα. Σύμφωνα με τις νομικές ρυθμίσεις, εάν είστε νομικό πρόσωπο, ενδέχεται να σας στέλνουμε email μάρκετινγκ χωρίς τη συγκατάθεσή σας. Σε αυτή την περίπτωση, μπορείτε επίσης να εξαιρεθείτε από τη λήψη email μάρκετινγκ από εμάς ανά πάσα στιγμή.

Πριν μοιραστούμε τα προσωπικά σας δεδομένα με τρίτους για τους σκοπούς μάρκετινγκ τους, θα ζητήσουμε τη ρητή συγκατάθεσή σας.

Μπορείτε να μας ζητήσετε να σταματήσουμε να σας στέλνουμε μηνύματα μάρκετινγκ ανά πάσα στιγμή συνδεόμενοι στον ιστότοπο και το προφίλ σας, και προσθέτοντας ή ακυρώνοντας τα αντίστοιχα πεδία/επιλογές για να προσαρμόσετε τις προτιμήσεις μάρκετινγκ σας.

Παρακαλούμε σημειώστε - εάν αρνηθείτε να λαμβάνετε μηνύματα μάρκετινγκ, αυτή η άρνηση δεν ισχύει για προσωπικά δεδομένα που παρέχονται ως αποτέλεσμα άλλων συναλλαγών όπως αγορές, εγγραφές εγγύησης κ.λπ.

Θα διατηρήσουμε τα προσωπικά σας δεδομένα για όσο χρόνο είναι απαραίτητο για την εκπλήρωση των σκοπών για τους οποίους τα συλλέξαμε, ή για την ικανοποίηση νομικών, λογιστικών ή απαιτήσεων αναφοράς.

Όταν αποφασίζουμε πόσο καιρό να αποθηκεύσουμε αυτά τα δεδομένα, εξετάζουμε την ποσότητα, τη φύση και την ευαισθησία τους, τον πιθανό κίνδυνο μη εξουσιοδοτημένης χρήσης ή αποκάλυψης και τους σκοπούς της επεξεργασίας, καθώς και το αν μπορούν να επιτευχθούν με άλλα μέσα και τις νομικές απαιτήσεις.

Παρακαλούμε σημειώστε - οι φορολογικοί νόμοι απαιτούν από εμάς να διατηρούμε βασικές πληροφορίες σχετικά με τους πελάτες μας (όπως στοιχεία επικοινωνίας, ταυτότητα, οικονομικά δεδομένα και δεδομένα συναλλαγών) για έως και δέκα χρόνια μετά την παύση της ιδιότητάς σας ως πελάτη μας.

Υπό ορισμένες συνθήκες, ενδέχεται να ανωνυμοποιήσουμε τα προσωπικά σας δεδομένα για ερευνητικούς ή στατιστικούς σκοπούς, οπότε ενδέχεται να χρησιμοποιήσουμε αυτές τις πληροφορίες επ' αόριστον και χωρίς να σας ειδοποιήσουμε.

10. Σύνδεσμοι Τρίτων

Αυτός ο ιστότοπος ενδέχεται να περιλαμβάνει συνδέσμους προς άλλους ιστότοπους, πρόσθετα και εφαρμογές τρίτων. Κάνοντας κλικ σε αυτούς τους συνδέσμους ή ενεργοποιώντας τους ενδέχεται να επιτρέψετε σε τρίτους να συλλέξουν ή να μοιραστούν δεδομένα για εσάς. Δεν ελέγχουμε αυτούς τους ιστότοπους και επομένως δεν είμαστε υπεύθυνοι για τις δηλώσεις απορρήτου τους. Όταν αποχωρείτε από τον ιστότοπό μας, συνιστούμε να διαβάζετε τη δήλωση απορρήτου κάθε ιστότοπου που επισκέπτεστε.

11. Cookies

Μπορείτε να ρυθμίσετε το πρόγραμμα περιήγησής σας να αρνείται όλα ή ορισμένα cookies ή να σας προειδοποιεί όταν ιστότοποι ορίζουν ή έχουν πρόσβαση σε cookies. Εάν απενεργοποιήσετε ή αρνηθείτε τα cookies, παρακαλούμε σημειώστε ότι ορισμένα μέρη αυτού του ιστότοπου ενδέχεται να καταστούν μη προσβάσιμα ή να μην λειτουργούν σωστά.
